# 查找您的 MCID

如需查找您的 Microsoft Certification identification number (MCID)，请登录您的 [Certification Dashboard.](https://aka.ms/certdashboard)。 您的 MCID在该页面的右上角。

![MCID 在该页面的右上](images/certification-dashboard-find-mcid.png)
